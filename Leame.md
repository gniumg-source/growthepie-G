Bienvenido a growthepie 📏🥧
Logotipo de growthepie.com en blanco
growthepie es la plataforma de análisis abierta para el ecosistema Ethereum. Desde la red principal hasta las capas 2 y las aplicaciones en cadena, mostramos el crecimiento del ecosistema a la vez que empoderamos a los desarrolladores de Ethereum con métricas abiertas y prácticas sobre el uso y la adopción.

gtp - ADN
El gtp-dnarepositorio contiene las definiciones de datos principales y los archivos de mapeo que impulsan growthepie.com . Si forma parte de un equipo de cadena o aplicación, puede modificar los metadatos de la cadena aquí o cargar los logotipos de la aplicación.

gtp-backend - Repositorio de backend
El gtp-backendrepositorio sirve como columna vertebral para el panel de growthepie, brindando canales de datos robustos y curación de datos.

Características
Canalizaciones de datos desde RPC y otras fuentes de datos a nuestra base de datos PostgreSQL
Limpieza y curación de datos
Programación mediante Airflow
gtp-frontend - Implementación de frontend
El gtp-frontendrepositorio contiene el código frontend para la plataforma growthepie, diseñado para ofrecer una experiencia de usuario rápida y receptiva.

Características
Visualización dinámica de datos con una variedad de opciones de filtrado fáciles de usar, que mejoran la exploración y el análisis de datos.
Funciones de categorización y comparación de datos para obtener información clara sobre las tendencias de los datos
Filosofía de diseño consistente para una interfaz de usuario fácil de usar
Wiki - Base de conocimientos
Nuestro wikicontenido incluye conocimientos seleccionados destinados a educar a los usuarios, desarrolladores e inversores sobre el espacio L2, fomentando un ecosistema transparente.

Características
Contenido educativo sobre diferentes soluciones de escalamiento
Documentación de la API
Explicaciones completas sobre cómo conservamos nuestros datos
